# Security Policy

## Scope

This repository defines the Quantum-Safe Agentic Infrastructure framework, including reference architecture, roadmap, and related security principles.

It does not currently contain production cryptographic implementation code.

## Reporting Security Issues

If you discover a security issue related to one of the referenced implementation projects, please report it through the relevant project repository.

For general security concerns related to this framework, open a private discussion or contact the maintainer.

## Security Focus

This framework focuses on:

- Post-quantum cryptography readiness
- Agentic AI security
- Secure key management
- Reproducible systems engineering
- Distributed systems reliability
- Blast-radius analysis
