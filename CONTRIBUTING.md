# Contributing

Contributions are welcome.

This repository is focused on the development of Quantum-Safe Agentic Infrastructure as a practical framework for securing AI-driven, post-quantum, and distributed digital systems.

## Areas for Contribution

- Reference architecture improvements
- Threat model additions
- Post-quantum cryptography use cases
- Agentic AI security scenarios
- Distributed systems reliability patterns
- Reproducible engineering practices
- Documentation improvements

## Contribution Guidelines

Please keep contributions aligned with the mission of the framework:

> To build secure infrastructure for a future where AI agents, quantum threats, and distributed systems converge.

Use clear technical language, avoid hype, and prefer practical implementation guidance.
